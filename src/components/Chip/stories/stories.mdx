# Chip
Chips are compact elements that represent an input, attribute, or action.

## How to use?

```
import Chip from './components/Chip';

const chipConfig = { 
        bgColor: colors.LIGHT_RED,
    color: colors.DARK_RED,
    value: "Deleted"
}

<Chip {...chipConfig}/>
```
